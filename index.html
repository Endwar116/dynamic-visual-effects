📖 使用方法

方法 1：本地預覽（推薦）
將所有 HTML 檔案保存到本地，直接用瀏覽器雙擊打開，無需伺服器。

方法 2：部署到 Render/GitHub Pages
將檔案上傳到 GitHub 倉庫，透過 Render 或 GitHub Pages 部署，獲得公開 URL。

方法 3：整合到 React/Vue 專案
複製 Canvas 初始化和繪製邏輯到你的框架，使用 useEffect/mounted hook 管理生命週期。

方法 4：提供給其他 AI
每個檔案頂部包含完整的技術規格註解，其他 AI 可直接讀取並理解實現邏輯。
